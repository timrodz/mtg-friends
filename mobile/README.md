# Tie Breaker Mobile App

## Pre-requisites

Deploy app w/ EAS:

```shell
npm install -g eas-cli
```

## Builds

### iOS - Local

```shell
npm run build:ios:local
```

1. Setup the expo environment: https://docs.expo.dev/get-started/set-up-your-environment/?mode=development-build&platform=ios&device=physical&buildEnv=local
2. Setup code signing: https://github.com/expo/fyi/blob/main/setup-xcode-signing.md
3. Note: the first time you build locally
